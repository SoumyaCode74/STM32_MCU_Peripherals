void I2C_Init ( I2C_Handle_t * pI2CHandle)

Initialize the I2C peripheral module based on the SCLK frequency, Device address (if it is a slave), Acknowledgement enable, and Fast Mode Duty Cycle.

Parameters
I2C_Handle_t* Pointer to instance of class I2C_Handle_t
Return values
None

< Variable to store the PCLK source

< Variable to store the SCLK speed setting to FREQ bits of CR2 register

< Variable to store the CCR value to write to the CCR bits of CCR register

< Variable to store the current configured PCLK frequency

< Disable the I2C peripheral

< Configure the mode (Standard or Fast)

< Configure the device address (if the device is slave)

< Enable the Acking

< Configure the rise time for I2C pins
